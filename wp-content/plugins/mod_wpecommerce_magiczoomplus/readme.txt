INSTALLATION:

1. Unzip the files, keeping the folder structure intact.

2. Create folder 'magiczoomplus-wpecommerce' in your "/wp-content/plugins" directory.

3. Upload the 'magiczoomplus-wpecommerce' folder to your "/wp-content/plugins/magiczoomplus-wpecommerce" directory.

4. Upload the 'magiczoomplus-wpecommerce.php' file to your "/wp-content/plugins/magiczoomplus-wpecommerce" directory.

5. Activate the plugin through the 'Plugins' menu in WordPress.

6. You've now installed the demo version of Magic Zoom Plus!

7. To upgrade, buy Magic Zoom Plus and overwrite the magiczoomplus.js file with the same file from the full version.
